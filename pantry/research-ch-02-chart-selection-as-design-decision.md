## 2. Core Concept — State of the Field

Chart selection is not a software-default choice. It is a chain from reader question to data structure to comparison type to visual form.

The FT Visual Vocabulary is useful as a navigation device, not a rulebook. Cairo's contribution is the ethical frame: a chart can be morally wrong when it predictably impedes understanding.

## 3. Application Domain Examples

1. Funding by sector: bar chart versus pie chart.
2. Trend over time: line chart versus stacked area.
3. Geographic rate: choropleth versus dot map.
4. Relationship: scatterplot versus connected line.
5. Composition: stacked bar versus treemap.

## 4. Book's Thesis Connection

Claude can implement a requested chart, but the human must choose the form. This chapter trains the reader to specify chart type as a defensible design decision.
