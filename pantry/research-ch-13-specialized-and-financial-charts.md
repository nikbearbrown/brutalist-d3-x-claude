## 2. Core Concept — State of the Field

Domain charts encode specialized questions. Candlesticks compress open, high, low, and close into a single glyph. Kagi and Point & Figure remove ordinary time spacing to emphasize trend reversal and price movement.

Bullet graphs improve on gauges because position on a common scale is more accurate than angle. Radar charts remain risky because axis order changes the perceived shape.

## 3. Application Domain Examples

1. Candlestick/OHLC for financial price action.
2. Kagi and Point & Figure for reversal and support/resistance questions.
3. Bullet graph for actual, target, and qualitative bands.
4. Radar chart for profile comparison when axis order is meaningful and disclosed.
5. Domain-specific rose charts where the convention itself carries meaning.
